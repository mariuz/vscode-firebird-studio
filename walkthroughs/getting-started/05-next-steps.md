## Next Steps

- **Use IntelliSense** — start typing a table or column name in a `.sql` file for context-aware completions.
- **Use snippets** — type `fb` in a `.sql` file and press `Tab` to expand a code snippet.
- **Generate mock data** — right-click a table and choose **Generate Mock Data** (requires a [Mockaroo API key](https://www.mockaroo.com/)).
- **Ask Copilot** — with GitHub Copilot Chat installed, use `@firebird` for natural-language queries, `/optimize`, and `/explain`.
- **Customize settings** — open Settings (`Ctrl+,`) and search `firebird` to adjust logging level, pagination, and more.
