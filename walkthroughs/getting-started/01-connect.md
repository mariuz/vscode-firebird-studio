## Add Your First Connection

Click the **Firebird** icon in the Activity Bar to open the **DB Explorer**, then add a connection:

1. Click the **+** (Add New Connection) icon in the DB Explorer title bar.
2. Fill in host, port, database path, username, and password (defaults to `localhost` / `3050` / `SYSDBA` / `masterkey` for a fresh install).
3. Click **Connect**.

No Firebird server handy? [Download Firebird](https://firebirdsql.org/en/firebird-downloads/) — the installer bundles the `employee.fdb` sample database, or point this at an embedded `.fdb`/`.gdb` file directly.
