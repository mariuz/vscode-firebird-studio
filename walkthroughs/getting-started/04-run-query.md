## Run Your First Query

1. Create a new SQL file: right-click the database node and choose **New Query**, or open a new file and save it with a `.sql` extension.
2. Type a query, e.g. `SELECT FIRST 10 * FROM EMPLOYEE`.
3. Press `Ctrl+Alt+Q` (or right-click the editor and choose **Run Firebird Query**).

Results open in a new panel with pagination, sorting, filtering, and chart visualization.

[Open a New Query](command:firebird.explorer.newSqlDocument)
